# homelab

My Ubuntu Server 22.04 LTS Homelab

## Info

Create and manage luks2 encrypted files.

The following scripts store the encrypted data in <b>~/data</b> and mount it to <b>~/container</b>.

Use at your own risk.

## Scripts

create-image.sh: Create a luks2 encrypted image file and mount it\
mount-image.sh: Mount and unlock an existing image\
umount-image.sh: Umount and close
